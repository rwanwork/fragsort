Files in this directory and their purpose. Files with a * next to their name are part of the distribution.

Bash scripts

call-sge.sh
Main script that is executed (once) on chimera. It calls run-sge.sh multiple times.
dist.sh
Script to send files to chimera. Both FragSort and DNA-Re-Pair are sent.
sge-run-expt.sh
A script specific for chimera that is used to handle paths and program arguments and pass them to run-expt.pl.
sge-run-index-expt.sh
A script specific for chimera that is used to handle paths and program arguments and pass them to run-index-expt.pl.

Perl scripts

fix-timefiles.pl
Change the timing files into a tabular format for easier processing.
make-xml.pl
Convert a FASTQ data file into a simple XML file.
parse-stdout.pl
Parse the standard out files generated by the SGE system (which are of the form sge-run-expt.sh.o or sge-run-index-expt.sh.o) and produce a tabular output which is easier to see. Useful to get an idea of the progress of the experiments. Experiments with an asterisk are currently running.
reorder.pl
Script used to reorder a FASTQ data file using an "ordering file" as a guide. This "ordering file" is in binary format and produced by fragsort.
run-expt.pl
Main script for running the non-indexing experiments that can be used on any machine (home machine, chimera, etc.).
run-index-expt.pl
Main script for running the indexing experiments that can be used on any machine (home machine, chimera, etc.).
* separate.pl
Script used to separate the lines of a FASTQ data file. An assumption is made that each sequence and quality scores entry is all on one line.
* run-fragsort.pl
Adapted version of run-expt.pl for distribution to users. Not used for the paper.
* run-fragsort-index.pl
Adapted version of run-expt-index.pl for distribution to users. Not used for the paper.
* collect-stats.pl
Adapted version of identifier-count.pl for distribution to users. Not used for the paper.

Perl modules

* help_fragsort.pm
Miscellaneous functions used by the run-*.pl scripts. Includes wrapper functions that calls compression routines (gzip, bzip, gamma coding, and delta coding) and reports on compression effectiveness. Also checks if compression was successful by comparing their md5sum signatures.
analyze_fragsort.pm
Analysis functions that are used by parse-stdout.pl and fix-timefiles.pl.
* bitcoding_binary.pm
No bit coding (i.e., binary coding) for indexed scheme. Not used for the paper.
bitcoding_deltagamma.pm
Bit coding using delta and gamma coding for indexed scheme.
